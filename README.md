# Team Agency Portfolio

A simple portfolio website for a student team built with HTML5, CSS3, and JavaScript.

## Project structure

- `index.html` - Main page markup
- `css/style.css` - Main styles
- `css/themes.css` - Light/dark theme styles
- `css/responsive.css` - Responsive layout rules
- `js/data.js` - Data for team members and projects
- `js/render.js` - Render dynamic content
- `js/theme.js` - Theme toggle logic
- `js/contact.js` - Contact form validation

## Notes

- No frameworks or libraries are used.
- The site is built to be easy to explain in a student project demonstration.
