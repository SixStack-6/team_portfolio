# Team Agency Portfolio

A responsive portfolio website developed as part of a JavaScript Capstone Project. The website introduces the team members, showcases completed projects, highlights technical skills, and provides a contact form with client-side validation. The project is built entirely with **HTML5**, **CSS3**, and **JavaScript** without using any external frameworks or libraries.

---

## Features

* Responsive design for desktop, tablet, and mobile devices
* Dynamic rendering of team members using JavaScript
* Projects showcase section
* Skills section
* About Team section
* Contact form with JavaScript validation
* Smooth scrolling navigation
* Clean and modern user interface

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)


2. Open the project folder.

```bash

```

3. Launch the project by opening `index.html` in your browser.
